### Navigation & Search
| Action | Shortcut |
| :--- | :--- |
| Find whole project | `CTRL` + `SHIFT` + `F` |
| Find/replace whole project | `CTRL` + `SHIFT` + `R` |
| Find Classes | `SHIFT` (2x) |
| Find Actions | `CTRL` + `ALT` + `A` |
| Browse Forward/Backward | `CTRL` + `ALT` + `Arrow` |

### Refactoring
| Action | Shortcut |
| :--- | :--- |
| Change Method Signature | `CTRL` + `F6` |
| Extract Method | `CTRL` + `ALT` + `M` |
| Extract Parameter | `CTRL` + `ALT` + `P` |
| Extract Variable | `CTRL` + `ALT` + `V` |
| Access Method | `CTRL` + `B` |
| Access Implementations | `CTRL` + `ALT` + `B` |

### View
* **Zoom in:** `⌃Ctrl` `⌥Option` `=`
* **Zoom out:** `⌃Ctrl` `⌥Option` `-`
* **Reset to default:** `⌃Ctrl` `⌥Option` `0`
