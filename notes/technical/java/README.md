### Useful libs

- **ArchUnit**: Java architecture test library to specify and assert architectural rules.
- **Bucket4j**: Java rate-limiting library based on the token-bucket algorithm.
- **Resilience4j**: Lightweight fault tolerance library designed for functional programming.
- **Swagger**: Toolset for OpenAPI specification to design, build, and document APIs.
- **Redisson**: Redis Java client with features of an in-memory data grid.
- **Kryo**: Fast and efficient binary object graph serialization framework for Java.
- **Lombok**: Library that provides annotations to reduce boilerplate code like getters and setters.
- **Feign**: Declarative REST web service client that makes writing web service clients easier.
- **Testcontainers**: Java library that supports JUnit tests, providing lightweight, throwaway instances of common databases or anything else that can run in a Docker container.
- **Microstreams**: High-performance Java native object graph storage engine.
- **Gatling**: Powerful open-source load and performance testing tool for web applications.
- **Apache StringUtils**: Part of the Commons Lang library, providing helper methods for String manipulation.
